# KonradBank

Simulador de cajero automático desarrollado como proyecto académico para la materia **Técnicas de Programación II** en la Fundación Universitaria Konrad Lorenz.

> ⚠️ **Nota**: Este es un proyecto educativo, no es una aplicación bancaria real.

## Descripción

Aplicación de escritorio en Java Swing que simula operaciones básicas de un cajero automático: depósitos y retiros.

## Tecnologías

- Java 11+
- Java Swing (GUI)
- Eclipse IDE
- Arquitectura MVC

## Ejecución

1. Importar el proyecto en **Eclipse**
2. Ejecutar `AplMain.java` en `src/co/edu/konradlorenz/controller/`
